Como exportar para JPG / PNG: os logos são SVG vetoriais (escalam para qualquer tamanho sem perder qualidade). Para um JPG/PNG rápido: clica com o botão direito numa imagem → Abrir imagem em novo separador, ou abre o ficheiro .svg da pasta brand/ e usa Exportar / Print → PDF. Para tamanho exato (ex.: 1080×1080 para Instagram), abre o .svg e ajusta width/height, ou importa no Figma/Canva/Illustrator.
