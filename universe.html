⌨ Keys & Shortcuts — Free Timeline Universe

Navigation

Scroll wheel Zoom in / out at cursor
Alt + Scroll Pan left / right
Shift + Scroll Pan up / down
Scroll on panel Resize universe rows
Drag universe label Move a universe row up or down
Click & drag Pan the timeline freely
+ Zoom in
- Zoom out
0 Show full 150.000 BC → 20.000 AC range
← → ↑ ↓ Pan with arrow keys

Events

Double-click Add event at that year & track
Click event Open event details / edit

Undo & Redo

Ctrl Z Undo last action (up to 60 steps)
Ctrl Y Redo (also Ctrl Shift Z)

Universe Panel Buttons

hide / show Dim or restore a universe track
edit Rename, recolour, add notes
info Browse all events & connections
del Delete universe and all its events

Toolbar Buttons

+ Universe Add a new universe / timeline track
+ Event Add a new event to a universe
+ Connection Link two events with an arc
+ Character Create a character record
Categories Manage & filter event categories
Stats View timeline statistics overview
↩ Undo / ↪ Redo Step back or forward through history
Import Load a previously exported JSON file
Export JSON Save full data as a JSON backup
Save HTML Export a self-contained offline page

Other Shortcuts

S Toggle stats panel
Esc Close any open panel or modal
? Toggle this help panel
