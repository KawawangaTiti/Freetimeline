Navigation

Scroll wheel Zoom in / out at cursor
Alt + Scroll Pan left / right
Shift + Scroll Pan up / down

- Zoom out

Events

Double-click Add event at that year & track
Click event Open event details / edit

Undo & Redo

Ctrl Z Undo last action (up to 60 steps)
Ctrl Y Redo (also Ctrl Shift Z)

Universe Panel Buttons

hide / show Dim or restore a universe track
edit Rename, recolour, add notes
info Browse all events & connections
del Delete universe and all its events

+ Universe Add a new universe / timeline track
